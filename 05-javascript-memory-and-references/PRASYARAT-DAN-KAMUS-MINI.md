## Kamus Mini Memory
- Primitive: nilai yang disalin sebagai value.
- Reference: acuan ke object/array/function di heap.
- Reachability: objek masih bisa dijangkau dari root reference.
- Referential equality: dua variabel menunjuk reference yang sama.
- Memory leak: objek tetap reachable walau tidak diperlukan secara bisnis.
