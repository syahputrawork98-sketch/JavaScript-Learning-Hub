## Exit Criteria Buku 06
- Bisa memprediksi output snippet runtime sederhana tanpa trial-and-error.
- Bisa menjelaskan binding `this` berdasarkan call-site.
- Bisa membedakan urutan eksekusi sync, microtask, dan macrotask.
- Bisa mengidentifikasi bug mutation akibat shared reference.
- Bisa melacak alur error pada Promise chain dan `async/await`.
- Bisa menggabungkan beberapa konsep runtime saat debugging kasus campuran.
