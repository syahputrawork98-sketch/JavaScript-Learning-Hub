# Content Map - 01 JavaScript Tutorial

## Bab 1 - Fondasi JavaScript
### In Scope
- Variabel: `let`, `const`.
- Tipe data dasar: string, number, boolean, null, undefined.
- Operator dasar dan perbandingan.
- Kontrol alur: `if/else`, `switch`.

### Out of Scope
- Detail coercion yang kompleks.
- Bahasan engine-level.

### Output Bab
- Program keputusan sederhana (contoh: cek kelulusan, diskon, akses role).

## Bab 2 - Function dan Modular Thinking
### In Scope
- Function declaration dan expression.
- Parameter, argument, return.
- Scope dasar function.

### Out of Scope
- Closure mendalam.
- Functional programming lanjut.

### Output Bab
- Sekumpulan fungsi kecil reusable untuk kalkulasi/validasi sederhana.

## Bab 3 - Array dan Object
### In Scope
- Create/read/update data.
- Method dasar (`push`, `map`, `filter`, `find`).
- Object literal dan nested object.

### Out of Scope
- Optimasi performa data structure tingkat lanjut.

### Output Bab
- Pengelolaan data list sederhana (contoh: daftar tugas atau produk).

## Bab 4 - Loop dan Problem Solving Dasar
### In Scope
- `for`, `while`, `for...of`.
- Iterasi dengan kondisi berhenti yang benar.
- Pola pemrosesan list dasar.

### Out of Scope
- Algoritma kompleks dan optimasi Big-O mendalam.

### Output Bab
- Solusi otomatis untuk proses berulang dengan validasi hasil.

## Bab 5 - Async Dasar
### In Scope
- `setTimeout` untuk memahami delay.
- Promise dasar (`then/catch`).
- `async/await` dan error handling dasar.

### Out of Scope
- Event loop internals detail.
- Concurrency control lanjutan.

### Output Bab
- Alur ambil data sederhana async dan tampilkan status loading/error.

## Bab 6 - Mini Project
### In Scope
- Integrasi konsep bab 1-5.
- Struktur kode sederhana tapi rapi.
- Validasi input + handling error dasar.

### Out of Scope
- Arsitektur enterprise.
- Integrasi framework kompleks.

### Output Bab
- 1 mini app selesai dengan checklist fitur minimum.
