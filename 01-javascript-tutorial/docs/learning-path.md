# Learning Path - 01 JavaScript Tutorial

## Tujuan Track
Membawa learner dari nol sampai bisa membuat mini app JavaScript sederhana sambil paham alasan teknis di balik keputusan kodenya.

## Prasyarat
- Bisa menggunakan editor kode.
- Paham konsep file/folder dasar.
- Sudah pernah menjalankan `node` atau JavaScript di browser (minimal sekali).

## Alur Bab
1. Bab 1 - Fondasi JavaScript
- Materi: variabel, tipe data, operator, `if/else`, `switch`.
- Target: bisa menulis logika keputusan sederhana dengan benar.
- Estimasi: 3-4 jam.

2. Bab 2 - Function dan Modular Thinking
- Materi: declaration/expression, parameter, return, scope dasar.
- Target: bisa memecah masalah jadi fungsi kecil.
- Estimasi: 3-4 jam.

3. Bab 3 - Array dan Object
- Materi: CRUD data sederhana, method dasar, nested data.
- Target: bisa memodelkan data nyata ke struktur JS.
- Estimasi: 4-5 jam.

4. Bab 4 - Loop dan Problem Solving Dasar
- Materi: `for`, `while`, `for...of`, pola iterasi.
- Target: bisa otomatisasi proses berulang tanpa bug dasar.
- Estimasi: 3-4 jam.

5. Bab 5 - Async Dasar
- Materi: `setTimeout`, Promise, `async/await`, `try/catch` dasar.
- Target: paham alur tunggu hasil dan error async sederhana.
- Estimasi: 4-5 jam.

6. Bab 6 - Mini Project
- Materi: gabungan konsep bab 1-5.
- Target: menyelesaikan mini app end-to-end.
- Estimasi: 5-7 jam.

## Checkpoint Per Bab
Setiap bab dinyatakan lulus jika learner bisa:
- Menjelaskan konsep dengan bahasa sendiri.
- Memprediksi output contoh inti sebelum menjalankan kode.
- Menyelesaikan minimal 80% latihan bab.
- Memperbaiki minimal 1 bug dasar dari studi kasus bab.

## Exit Criteria Track
- Bisa membangun mini app JavaScript sederhana dari nol.
- Bisa membaca dan memperbaiki bug dasar tanpa bantuan intensif.
- Bisa menjelaskan sebab-akibat teknis keputusan coding utamanya.
