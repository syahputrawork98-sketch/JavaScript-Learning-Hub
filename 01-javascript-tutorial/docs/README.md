# Docs Index - 01 JavaScript Tutorial

Dokumen di folder ini adalah panduan operasional untuk membangun track `01-javascript-tutorial` sebagai "buku belajar" dari nol.

## Daftar Dokumen
- `learning-path.md`: urutan belajar bab 1-6, durasi, dan checkpoint.
- `content-map.md`: peta isi detail per bab (in-scope/out-of-scope mikro).
- `chapter-template.md`: template baku penulisan satu bab.
- `style-guide.md`: aturan bahasa, visual, dan kualitas materi.
- `changelog.md`: catatan perubahan track docs.
- `chapters/README.md`: index materi bab.
- `chapters/01-fondasi-javascript.md`: materi lengkap Bab 1.
- `chapters/02-function-dan-modular-thinking.md`: materi lengkap Bab 2.
- `chapters/03-array-dan-object.md`: materi lengkap Bab 3.
- `chapters/04-loop-dan-problem-solving-dasar.md`: materi lengkap Bab 4.
- `chapters/05-async-dasar.md`: materi lengkap Bab 5.
- `chapters/06-mini-project.md`: materi lengkap Bab 6.

## Urutan Pakai
1. Baca `learning-path.md` untuk lihat alur menyeluruh.
2. Baca `content-map.md` untuk tahu batas isi per bab.
3. Tulis materi menggunakan `chapter-template.md`.
4. Cek kualitas akhir dengan `style-guide.md`.
5. Catat update penting di `changelog.md`.

## Prinsip Inti
- Detail tapi tidak bertele-tele.
- Bahasa sehari-hari, tetap presisi teknis.
- Selalu jelaskan sebab-akibat, bukan hafalan sintaks.
