# Dokumentasi Buku 01

Folder `docs/` berisi panduan editorial untuk buku `01-javascript-tutorial`.

Dokumen di folder ini tidak dipakai sebagai materi belajar utama. Materi utama ada di folder `chapters/`.

## Tujuan

- menjaga struktur buku tetap konsisten
- menjaga batas topik tetap sesuai scope buku 01
- menyediakan template standar untuk menulis bab
- memudahkan penambahan materi baru tanpa mengubah gaya buku

## Daftar Dokumen

- `writing-rules.md`: aturan penulisan khusus buku 01
- `content-structure.md`: struktur isi, urutan bab, dan batasan tiap bab
- `topic-template.md`: template standar file bab

## Prinsip Pemisahan

- `docs/` hanya untuk aturan dan struktur
- `chapters/` untuk materi pembelajaran
- `examples/` untuk contoh kode terpisah
- `assets/` untuk diagram atau ilustrasi pendukung
