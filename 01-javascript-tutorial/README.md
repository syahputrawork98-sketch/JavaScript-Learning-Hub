# 01 - JavaScript Tutorial

Buku ini adalah pintu masuk untuk belajar JavaScript dari nol.

Fokus buku 01 adalah penggunaan bahasa: menulis kode, menyusun logika dasar, dan membangun program sederhana.

## Tujuan

- memahami sintaks dasar JavaScript
- menulis program kecil yang bisa dijalankan
- membangun fondasi sebelum masuk ke buku runtime dan model internal

## Cakupan

- program pertama
- variables
- data types
- operators
- conditionals
- loops
- functions
- arrays
- objects
- modules
- error handling

## Out of Scope

- runtime internals (execution context, closure internals, event loop detail)
- object internals mendalam
- engine internals dan optimasi

## Struktur Konten

- `chapters/`: materi utama per bab
- `examples/`: contoh kode tambahan
- `assets/`: diagram pendukung
- `docs/`: aturan editorial dan template penulisan
