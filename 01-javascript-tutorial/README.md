# JavaScript Tutorial

Track ini untuk mempercepat onboarding JavaScript dari nol sampai bisa membuat program kecil yang berjalan.

## Tujuan
- Memahami sintaks dan flow dasar JavaScript.
- Bisa memakai function, object, array, dan async dasar.
- Menyelesaikan mini app sederhana.

## Scope
- Variabel, tipe data, operator, kontrol alur.
- Function, object, array, loop.
- Async dasar (`setTimeout`, Promise, `async/await`) untuk use case sederhana.
- Latihan kecil per topik + mini project.

## Out of Scope
- Internals runtime yang detail.
- Framework/library spesifik.

## Exit Criteria
- Bisa membangun mini app JavaScript sederhana dari nol.
- Bisa membaca dan memperbaiki bug dasar tanpa bantuan intensif.
