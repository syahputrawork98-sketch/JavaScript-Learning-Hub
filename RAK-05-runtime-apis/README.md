# Rak 05: Modern Runtime APIs

*Level: Senior/Architect*

Rak ini menjelajahi ekosistem eksekusi JavaScript modern di luar browser. Kita akan membedah bagaimana Node.js, Deno, dan Bun menangani I/O, keamanan, dan performa untuk skala server.

## 🎯 Tujuan Rak Ini
- Membedah arsitektur internal Node.js (Event Loop & Libuv).
- Memahami evolusi sistem modul (CJS vs ESM).
- Mengeksplorasi inovasi keamanan Deno dan kecepatan ekstrim Bun.

- **[SR-01: Node.js Core](./SR-01_NodeJS/README.md)**: Fondasi runtime server-side paling dominan.
- **[SR-02: Bun & Deno Ecosystem](./SR-02_BunDeno/README.md)**: Masa depan runtime dengan fokus pada keamanan dan kecepatan.
