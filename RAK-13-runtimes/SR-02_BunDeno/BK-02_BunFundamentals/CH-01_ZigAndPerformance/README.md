# CH-01: Zig & Performance (The Speed Engine)

Bun ditulis menggunakan bahasa pemrograman **Zig**, yang memungkinkan kontrol memori tingkat rendah tanpa overhead yang besar.

## 🚀 Mengapa Bun Begitu Cepat?
1. **Zig Language**: Lebih ringan dan efisien dibanding C++ yang digunakan Node.js.
2. **JavaScriptCore**: Menggunakan engine milik Safari yang dioptimalkan untuk startup time yang cepat (berbeda dengan V8 yang dioptimalkan untuk long-running processes).
3. **No Overhead**: Bun menggabungkan bundler, test runner, dan package manager ke dalam satu executable tunggal.
