# CH-01: Performance Comparison (Benchmarking)

Bagaimana perbandingan performa Node.js, Deno, dan Bun dalam dunia nyata?

## 📊 Hasil Benchmark (General)

| Fitur | Node.js | Deno | Bun |
| :--- | :--- | :--- | :--- |
| **Startup Time** | Lambat | Cepat | Sangat Cepat |
| **HTTP Requests/sec** | Menengah | Tinggi | Sangat Tinggi |
| **Memory Usage** | Menengah | Rendah | Sangat Rendah |
| **Package Install** | Lambat | Cepat (URL) | Sangat Cepat |

*Catatan: Performa dapat bervariasi tergantung pada beban kerja dan hardware.*
