# CH-02: Standard Library (Deno.core)

Deno hadir dengan library standar yang dikurasi dengan baik, menghindari fragmentasi ekosistem.

## 📦 Nilai Tambah Deno Std
- Dipelihara langsung oleh tim inti Deno.
- Diuji secara ketat untuk interoperabilitas.
- Mengikuti pola desain yang konsisten.

## 🧩 Contoh: File Handling
Deno menggunakan `Deno.readFile` (Return Promise) sebagai standar kelas satu.
