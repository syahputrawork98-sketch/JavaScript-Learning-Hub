# Runtime Exercises and Katas

Track ini berisi latihan untuk menguatkan ketepatan eksekusi teknis, bukan sekadar hafalan konsep.

## Tujuan
- Membuat reasoning runtime lebih konsisten.
- Menurunkan bug regresi lewat latihan terstruktur.
- Meningkatkan kecepatan debug untuk kasus nyata.

## Scope
- Prediksi output sync/async.
- Katas untuk event loop, race condition, state mutation, dan memory pitfalls.
- Latihan refactor dengan acceptance criteria.

## Out of Scope
- Teori panjang tanpa praktik.
- Latihan yang tidak punya rubrik evaluasi.

## Exit Criteria
- Konsisten lulus drill prediksi output.
- Bisa menjelaskan akar masalah bug sebelum mengubah kode.
- Bisa membuktikan perbaikan dengan test/checklist sederhana.
