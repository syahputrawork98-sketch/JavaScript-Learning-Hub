# Rak: Spec (ECMAScript Specification)

*Level: Expert/Scholar*

Rak ini didedikasikan untuk bedah murni klausa demi klausa dari **ECMA-262 Specification** secara formal. Di sini, kita tidak belajar "cara pakai", melainkan belajar "definisi hukum" di balik setiap perilaku bahasa JavaScript.

## Tujuan Rak Ini
- Menjadi jembatan antara bahasa praktis dengan bahasa formal spesifikasi.
- Mendekomposisi dokumen PDF/HTML Spec yang sangat panjang menjadi unit-unit narasi yang mudah dipahami.
- Membantu *Engineer* untuk bisa berkontribusi atau memahami proposal baru di TC39.
