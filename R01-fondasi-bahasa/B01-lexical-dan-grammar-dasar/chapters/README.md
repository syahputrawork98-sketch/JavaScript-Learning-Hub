# Index Bab B01

`B01` membangun fondasi lexical dan grammar JavaScript sebelum masuk values/types (`B02`).

## Daftar Bab

1. `C01-lexical-fundamentals.md`  
   Tujuan: memahami aturan dasar lexical level paling awal.
2. `C02-whitespace-line-terminator-dan-comments.md`  
   Tujuan: memahami elemen non-eksekusi yang memengaruhi pembacaan kode.
3. `C03-identifiers-dan-reserved-words.md`  
   Tujuan: menulis nama identifier valid dan menghindari kata terlarang.
4. `C04-literals-dasar.md`  
   Tujuan: mengenal literal number/string/boolean/null secara aman.
5. `C05-blocks-dan-struktur-file.md`  
   Tujuan: memahami statement block dan struktur file JavaScript dasar.
6. `C06-strict-mode-dasar.md`  
   Tujuan: memahami fungsi `"use strict"` dan dampak paling umum.
7. `C07-numeric-literals-lanjutan.md`  
   Tujuan: memahami variasi numeric literal (binary, octal, hex, separator, bigint).
8. `C08-string-literals-dan-escape-sequences.md`  
   Tujuan: memahami escape sequence dan jebakan string literal.
9. `C09-template-literals-dan-tagged-template-dasar.md`  
   Tujuan: memahami template literal lebih dalam, termasuk pengantar tagged template.
10. `C10-regular-expression-literals-dasar.md`  
   Tujuan: memahami regex literal sebagai bagian lexical source code.
11. `C11-automatic-semicolon-insertion-asi.md`  
   Tujuan: memahami aturan ASI dan edge case yang sering menyebabkan bug.
12. `C12-unicode-dan-identifiers-internasional.md`  
   Tujuan: memahami implikasi Unicode untuk source text dan identifier.
13. `C13-reserved-words-contextual-dan-future.md`  
   Tujuan: memahami contextual keywords dan kata cadangan masa depan.
14. `C14-hashbang-dan-host-text-conventions.md`  
   Tujuan: memahami hashbang serta konvensi source text lintas host.
15. `C15-lexical-goals-dan-ambiguity-dasar.md`  
   Tujuan: memahami konsep lexical goal symbols dan ambiguitas parsing dasar.
