# C02 - Learning Outcomes R01

## Tujuan

Bab ini menjelaskan hasil belajar yang harus tercapai setelah menyelesaikan R01.

## Outcome Inti (Terukur)

Setelah menyelesaikan R01, pembaca diharapkan mampu:

- menulis program JavaScript dasar dengan struktur yang rapi
- menggunakan values, types, operators, dan control flow dengan benar
- menulis fungsi sederhana dan memahami scope dasar
- mengelola data dasar dengan object/array dan built-in umum
- menerapkan error handling dasar
- memecah kode ke beberapa file menggunakan ES modules dasar

## Indikator Verifikasi Outcome

| Outcome | Indikator Minimal |
| --- | --- |
| Menulis program dasar rapi | Kode terbagi jelas ke fungsi dan file sesuai tanggung jawab |
| Values/types/operators/control flow | Tidak salah pada kasus dasar coercion, kondisi, dan loop |
| Functions dan scope dasar | Fungsi punya input-output jelas dan tidak konflik scope dasar |
| Object/array built-ins | Dapat mengolah list/data object dengan method dasar yang tepat |
| Error handling dasar | Menggunakan `try...catch` pada kasus validasi sederhana |
| ES modules dasar | Program multi-file berjalan dengan `import/export` dasar |

## Bukti Kelulusan yang Disarankan

- Satu mini project sederhana multi-file.
- Satu dokumen ringkas berisi miskonsepsi yang berhasil diperbaiki.
- Hasil checklist kelulusan pada `C06`.

## Outcome yang Belum Ditargetkan

Topik berikut bukan target R01:

- runtime internals mendalam
- async orchestration detail
- prototype internals
- engine optimization detail

## Kriteria Siap Naik Rak

Pembaca dianggap siap lanjut ke `R02` jika:

- outcome inti tercapai stabil di praktik
- indikator verifikasi minimum terpenuhi
- checklist `C06` memenuhi batas kelulusan
