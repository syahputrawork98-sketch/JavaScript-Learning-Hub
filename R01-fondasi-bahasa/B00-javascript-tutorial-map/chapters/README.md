# Index Bab B00

`B00` adalah buku peta belajar untuk rak `R01`.  
Fungsinya memastikan pembaca tahu urutan belajar, batas scope, dan target kelulusan sebelum masuk buku konsep.

## Daftar Bab

1. `C01-cara-pakai-r01.md`  
   Tujuan: menetapkan cara menggunakan rak R01 (alur minimum vs alur lengkap).
2. `C02-learning-outcomes-r01.md`  
   Tujuan: menjelaskan kemampuan yang harus dimiliki setelah menyelesaikan R01.
3. `C03-peta-buku-b01-b06.md`  
   Tujuan: memetakan fungsi setiap buku `B01-B06` dan prasyarat bacanya.
4. `C04-batas-scope-r01.md`  
   Tujuan: menjelaskan topik yang masuk R01 dan topik yang dipindahkan ke rak lain.
5. `C05-workflow-belajar.md`  
   Tujuan: memberikan pola belajar operasional (baca, praktik, review, checkpoint).
6. `C06-checklist-kelulusan-dan-next-step.md`  
   Tujuan: menyediakan checklist kelulusan R01 dan transisi ke R02.

## Urutan Baca

- Wajib berurutan dari `C01` sampai `C06`.
- Jika pembaca sudah familiar dengan JavaScript dasar, boleh lompat ke `C03` lalu kembali ke `C04-C06`.
