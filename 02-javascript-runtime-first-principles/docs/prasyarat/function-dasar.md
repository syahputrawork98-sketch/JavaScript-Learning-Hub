# Prasyarat: Function Dasar

## Tujuan
- Paham fungsi sebagai blok logika yang bisa dipanggil ulang.
- Paham parameter dan argument.

## Contoh Ringkas
```js
function greet(name) {
  return `Halo, ${name}`;
}

console.log(greet('Ari')); // Halo, Ari
```

## Bedah Singkat
1. `name` pada deklarasi adalah parameter.
2. `'Ari'` saat pemanggilan adalah argument.
3. `return` mengembalikan hasil dari function.

## Checklist Cepat
- [ ] Bisa membuat function sederhana.
- [ ] Bisa jelaskan beda parameter dan argument.
