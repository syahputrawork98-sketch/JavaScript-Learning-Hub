# Prasyarat: Object Dasar

## Tujuan
- Paham object berisi pasangan `key: value`.
- Bisa akses property dengan notasi titik.

## Contoh Ringkas
```js
const user = {
  name: 'Ari',
  age: 21,
};

console.log(user.name); // Ari
user.age = 22;
console.log(user.age); // 22
```

## Bedah Singkat
1. Object menyimpan data terstruktur.
2. Property diakses dengan `obj.key`.
3. Nilai property bisa diperbarui jika object mutable.
