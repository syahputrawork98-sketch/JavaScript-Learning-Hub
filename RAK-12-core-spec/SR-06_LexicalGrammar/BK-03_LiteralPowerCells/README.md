# Buku 03: Literal Power Cells (The Data Units)

Buku ini membedah bagaimana Hub mengekstrak nilai data statis langsung dari aliran sinyal. Unit-unit ini disebut **Literals**.

## 🏗️ Katalog Sel Daya

1.  **[Bab 01: Numeric Literals](./CH-01_NumericLiterals/)**: Format pengkodean angka (Decimal, Hex, Binary, Octal).
2.  **[Bab 02: String & Escape Sequences](./CH-02_StringAndEscape/)**: Pemrosesan teks dan karakter kontrol tersembunyi.
3.  **[Bab 03: Regex Literals](./CH-03_RegexLiterals/)**: Unit pencarian pola sinyal yang kompleks.
4.  **[Bab 04: Template Literals](./CH-04_TemplateLiterals/)**: Kontainer energi dinamis dengan dukungan interpolasi.

## 🔍 Arsitek Mindset
Setiap Literal adalah instruksi instan untuk mengisi memori Hub dengan nilai tertentu. Memahami format literal (seperti perbedaan `0x`, `0b`, dan `0o`) memudahkan Anda dalam mengoptimalkan komunikasi data tingkat rendah.
