# Buku 02: Universal Tokens (The Hub Components)

Buku ini membedah bagaimana aliran sinyal mentah dirangkai menjadi komponen-komponen dasar yang dapat dikenali. Di dalam spesifikasi, komponen ini disebut **Tokens**.

## 🏗️ Inventori Komponen

1.  **[Bab 01: Identifiers & Reserved Words](./CH-01_IdentifiersAndReserved/)**: Aturan penamaan unit dan daftar kata terlarang yang dipesan untuk fungsi internal Hub.
2.  **[Bab 02: Punctuators](./CH-02_Punctuators/)**: Sakelar dan pemisah sinyal (seperti `{`, `}`, `(`, `)`).
3.  **[Bab 03: Token Basics](./CH-03_TokenBasics/)**: Dasar-dasar klasifikasi token primer.

## 🔍 Insight Arsitek
Tokens adalah atom dari bahasa JavaScript. Tanpa pengenalan token yang tepat, Hub tidak akan pernah bisa membangun struktur logika yang lebih besar (seperti Expressions atau Statements).
