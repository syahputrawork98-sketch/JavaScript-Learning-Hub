# Buku 02: Command Links (The Navigation System)

Buku ini membedah sistem navigasi dan koneksi antar unit di dalam Hub. Ini adalah cara energi berpindah dari satu modul ke modul lainnya melalui akses properti dan pemanggilan fungsi.

## 🏗️ Sirkuit Navigasi

1.  **[Bab 01: Property Access & Member Exps](./CH-01_PropertyAccess/)**: Cara mengakses slot internal menggunakan notasi titik (`.`) atau kurung siku (`[]`).
2.  **[Bab 02: Execution Calls & New](./CH-02_ExecutionCalls/)**: Memicu unit penggerak (`[[Call]]`) atau membangun unit baru (`[[Construct]]`).

## 🔍 Arsitek Mindset
Navigasi yang presisi menentukan kecepatan aliran data. Memahami perbedaan antara akses properti statis dan dinamis memungkinkan Anda mengoptimalkan jalur transmisi energi dalam Grid.
