# Sub-Rak: 08 Statements and Declarations

Fase ini membedah unit eksekusi yang lebih besar. Membedah segala jenis *Statement* (If, While, For, Switch) dan deklarasi variabel tingkat dasar (`var`, `let`, `const`).
