# Buku 01: Control Flow

Buku ini membedah "Sistem Navigasi" JavaScript. Kita akan mempelajari bagaimana *Engine* mengelola alur eksekusi melalui blok kode, deklarasi variabel, percabangan, pengulangan, hingga penanganan error (Clause 14 pada ECMA-262).

## Mengapa Mempelajari Ini?
Program tanpa kontrol alur hanyalah daftar instruksi linear yang kaku. Dengan memahami *Statements* dan *Declarations*, Anda belajar bagaimana membuat program yang cerdas, mampu mengambil keputusan, mengulang tugas secara efisien, dan yang paling penting: tetap berdiri tegak saat terjadi kesalahan (*Exception Handling*).
