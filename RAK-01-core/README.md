# Rak: Core (The Brain Pillars)

*Level: Practitioner/Architect*

Rak ini merupakan jantung dari perpustakaan `JavaScript-Knowledge-Base`. Di sini, pilar-pilar bahasa JavaScript dibedah secara mendalam menggunakan pemetaan granular 12 Sub-Rak yang terinspirasi langsung dari struktur Clause spesifikasi **ECMA-262**.

## Daftar Sub-Rak (12 Tracks)

1. `01_notational_conventions/` (Clause 5)
2. `02_data_types_and_values/` (Clause 6)
3. `03_abstract_operations/` (Clause 7)
4. `04_executable_code_and_contexts/` (Clause 9)
5. `05_ordinary_and_exotic_objects/` (Clause 10)
6. `06_lexical_grammar/` (Clause 11-12)
7. `07_expressions_and_operators/` (Clause 13)
8. `08_statements_and_declarations/` (Clause 14)
9. `09_functions_and_classes/` (Clause 15)
10. `10_scripts_and_modules/` (Clause 16)
11. `11_standard_built_in_objects/` (Clause 18-28)
12. `12_memory_and_concurrency/` (Clause 29)

*Setiap Sub-Rak berisi Buku-buku tematik yang membedah topik secara naratif dan praktis.*
