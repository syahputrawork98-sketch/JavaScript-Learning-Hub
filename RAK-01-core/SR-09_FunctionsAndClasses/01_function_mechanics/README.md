# Buku 01: Function Mechanics

Buku ini membedah "Mesin Utama" atau "Unit Pekerja" di JavaScript: **Functions** dan **Classes**. Kita akan mempelajari anatomi fungsi, perilaku parameter, rahasia `this`, hingga bagaimana `class` hanyalah cara cantik untuk menulis *Prototype* (Clause 15 pada ECMA-262).

## Mengapa Mempelajari Ini?
Fungsi adalah warga kelas satu (*First-class Citizen*) di JavaScript. Memahami bagaimana fungsi bekerja di tingkat internal bukan sekadar tahu cara menulisnya, tapi tahu bagaimana memori dikelola, bagaimana `this` mengikat konteks, dan bagaimana pola desain berorientasi objek di JavaScript sebenarnya bekerja di bawah tenda.
