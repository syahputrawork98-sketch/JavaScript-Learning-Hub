# CH-03: Introducing Objects

## Daftar Chapter:

1. **[CH-01: Object Constructors](./SEC-01_ObjectConstructors/)**: Menggunakan "Cetak Biru" (Blueprints) untuk membangun perangkat.
2. **[CH-02: Prototypal Inheritance](./SEC-02_PrototypalInheritance/)**: Bagaimana energi (metode) diwariskan antar perangkat.
3. **[CH-03: Classes in JS](./SEC-03_ClassesInJS/)**: Menggunakan desain arsitektur modern (ES6 Classes).
4. **[CH-04: JSON Data](./SEC-04_JSONData/)**: Cara mentransmisikan data energi antar sistem.
