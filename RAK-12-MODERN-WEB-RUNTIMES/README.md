# Rak 12: Modern Web Runtimes

Menjelajahi ekosistem eksekusi JavaScript modern di luar browser.

## Struktur Sub-Rak
- **[SR-01-NODEJS](./SR-01-NODEJS/README.md)**: Dominasi server-side dan ekosistem NPM.
- **[SR-02-BUN-DENO](./SR-02-BUN-DENO/README.md)**: Inovasi runtime tercepat dan keamanan sejak awal (Native TS).

## JavaScript: The Kinetic Hub
JS bukan lagi bahasa "animasi tombol", tapi pusat transmisi web:
1. **Event-Driven**: Sangat efisien untuk menangani ribuan koneksi I/O sekaligus.
2. **Largest Ecosystem**: Akses ke jutaan library siap pakai di NPM.
3. **Ubiquity**: Satu bahasa untuk Frontend, Backend, dan Mobile.
