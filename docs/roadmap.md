# Roadmap JavaScript Learning Hub

## Misi
Menyediakan jalur belajar JavaScript yang runtut: dari bisa pakai, paham alasan teknis, sampai mampu membaca konsep dalam konteks spesifikasi resmi.

## Urutan Track (Arsitektur 7 Root)
1. `01-javascript-tutorial`
- Fokus: kemampuan implementasi cepat.
- Hasil: bisa membangun mini app JavaScript sederhana.

2. `02-javascript-runtime-first-principles`
- Fokus: execution model inti JavaScript.
- Hasil: bisa menjelaskan execution context, scope, closure, hoisting, `this`, stack, dan memory model dasar.

3. `03-asynchronous-javascript-model`
- Fokus: model async runtime.
- Hasil: bisa memprediksi urutan eksekusi event loop, task queue, microtask queue, Promise, dan `async/await`.

4. `04-javascript-object-model`
- Fokus: object mechanics.
- Hasil: bisa menjelaskan prototype chain, descriptors, constructors, `new`, dan class syntax.

5. `05-javascript-memory-and-references`
- Fokus: memory behavior dan references.
- Hasil: bisa mengidentifikasi bug mutation/reference dan memilih strategi immutability dasar.

6. `06-javascript-runtime-exercises`
- Fokus: penguatan lewat eksperimen terukur.
- Hasil: reasoning teknis konsisten pada closure edge cases, `this` binding, async order, dan mutation behavior.

7. `07-javascript-specification-companion`
- Fokus: koneksi konsep ke ECMAScript spec.
- Hasil: bisa membaca bagian spesifikasi terkait konsep yang sedang dipelajari.

## Milestone Global
- Milestone 1: selesai tutorial + 1 mini app.
- Milestone 2: selesai runtime first-principles + lulus drill prediksi execution flow.
- Milestone 3: selesai async/object/memory models + lulus studi kasus kombinasi.
- Milestone 4: selesai runtime exercises + checklist anti-regresi reasoning terpenuhi.
- Milestone 5: selesai specification companion + mampu cross-reference konsep ke spec.

## Prinsip Kurikulum
- Dari penggunaan ke pemahaman mekanisme.
- Setiap konsep harus diuji lewat latihan.
- Setiap perubahan materi harus punya alasan pedagogis yang jelas.
- Pemisahan track harus menjaga boundary konsep tetap tegas.

## Status Migrasi
- Roadmap ini menetapkan target arsitektur 7 track sebagai standar resmi.
- Migrasi konten dari struktur lama dilakukan bertahap untuk menghindari broken links.
