# Roadmap JavaScript Learning Hub

## Misi
Menyediakan jalur belajar JavaScript yang runtut: dari bisa pakai, paham alasan teknis, sampai stabil di eksekusi produksi.

## Urutan Track
1. `01-javascript-tutorial`
- Fokus: kemampuan implementasi cepat.
- Hasil: bisa membangun mini app JavaScript sederhana.

2. `02-javascript-first-principles`
- Fokus: mental model core runtime language.
- Hasil: bisa memprediksi perilaku kode dan debug secara sebab-akibat.

3. `03-javascript-advanced-patterns`
- Fokus: keputusan pattern untuk codebase nyata.
- Hasil: bisa menulis kode yang maintainable, terstruktur, dan minim regresi.

4. `04-javascript-exercises-katas`
- Fokus: penguatan lewat latihan terukur.
- Hasil: reasoning teknis konsisten saat menghadapi kasus baru.

## Milestone Global
- Milestone 1: selesai tutorial + 1 mini app.
- Milestone 2: selesai first-principles + lulus drill prediksi output.
- Milestone 3: selesai advanced-patterns + 3 studi kasus refactor.
- Milestone 4: selesai exercises/katas + checklist anti-regresi terpenuhi.

## Prinsip Kurikulum
- Dari penggunaan ke pemahaman mekanisme.
- Setiap konsep harus diuji lewat latihan.
- Setiap perubahan materi harus punya alasan pedagogis yang jelas.
