# Docs Index

Dokumentasi ini menjadi pengarah utama untuk pengembangan JavaScript Learning Hub.

## Daftar Dokumen
- `roadmap.md`: urutan belajar dan hasil akhir lintas track.
- `content-plan.md`: rencana isi per track, batas materi, dan prioritas penulisan.
- `operating-rules.md`: aturan editorial agar gaya, kualitas, dan struktur konsisten.
- `migration-map.md`: peta migrasi struktur lama ke arsitektur 7 track.

## Cara Pakai
1. Mulai dari `roadmap.md` untuk melihat jalur belajar global.
2. Lanjut ke `content-plan.md` saat menyusun/menambah materi.
3. Gunakan `migration-map.md` saat memindahkan konten lintas track.
4. Gunakan `operating-rules.md` sebagai checklist sebelum merge perubahan docs.
