# Dokumentasi Global v2

Folder ini berisi aturan tingkat **perpustakaan** (global).

Dokumen di sini tidak menyimpan materi JavaScript inti. Materi tetap berada di level buku.

## Isi Utama

- `repository-structure.md`: definisi struktur library -> rak -> buku -> bab.
- `track-map.md`: daftar rak resmi v2 dan alur belajarnya.
- `writing-rules.md`: baseline aturan penulisan yang wajib untuk semua buku.
- `contribution-rules.md`: alur perubahan struktur, materi, dan log perubahan.
- `templates/`: template untuk `docs/` di level rak dan buku.

## Prinsip Scope

- Root `docs/` untuk aturan global.
- `docs/` di rak untuk aturan yang hanya berlaku pada rak tersebut.
- `docs/` di buku untuk aturan khusus buku tersebut.
- Jika ada konflik aturan, prioritasnya: **buku -> rak -> root**.
