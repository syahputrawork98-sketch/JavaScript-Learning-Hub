# JavaScript Knowledge Base: Documentation Hub

Pusat dokumentasi ini menyimpan seluruh cetak biru, standar kualitas, dan panduan meta untuk membangun repository ini secara konsisten mengikuti standar MDN Web Docs.

## Daftar Dokumen Utama

1.  **[Architecture Standards](./standards/architecture.md)**: Aturan pewajiban file per tingkatan (Rak, Buku, Bab).
2.  **[Repository Plan](./repository-plan/README.md)**: Dekomposisi total dari MDN JavaScript ke dalam struktur 11-Rack.
3.  **[JavaScript Origins](./javascript-origins.md)**: Kisah legendaris pembuatan bahasa dalam 10 hari oleh Brendan Eich.
4.  **[JavaScript History](./javascript-history.md)**: Transformasi dari 10 hari pengodean hingga bahasa universal internet.
5.  **[Philosophy & Essence](./javascript-philosophy.md)**: Membedah Event Loop dan fleksibilitas tanpa batas.
6.  **[Why JavaScript?](./why-javascript.md)**: Mengapa tetap memilih JS di era persaingan bahasa modern.
7.  **[Structure Guide](./structure-guide.md)**: Panduan teknis pengorganisasian folder.
8.  **[Roadmap](./roadmap.md)**: Milestone pengembangan fitur.
