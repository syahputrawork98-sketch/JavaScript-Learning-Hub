# Sejarah & Evolusi JavaScript

> **"From a 10-day Prototype to the Language of the Internet."**

## 1. Kelahiran Kilat (1995)

JavaScript diciptakan oleh **Brendan Eich** di Netscape hanya dalam waktu **10 hari** pada Mei 1995. Awalnya diberi nama **Mocha**, kemudian **LiveScript**, dan akhirnya menjadi **JavaScript** sebagai strategi pemasaran untuk mendompleng popularitas bahasa Java pada saat itu.

### Latar Belakang Masalah
Pada era awal web, halaman hanya bersifat statis. Netscape ingin memberikan "perekat" (glue language) yang memungkinkan desainer web menambahkan interaktivitas sederhana (seperti validasi form) tanpa harus menguasai bahasa berat seperti Java atau C++.

## 2. Era Perang Browser & Standarisasi (1996 - 2008)

Netscape menyerahkan JavaScript ke **ECMA International** untuk distandarisasi, melahirkan spesifikasi **ECMAScript**.
- **JScript (Microsoft)**: Internet Explorer membuat versi tiruannya sendiri, memicu era di mana kode web harus ditulis berbeda untuk setiap browser.
- **The Dark Ages**: JavaScript sempat dianggap sebagai bahasa "mainan" yang lambat dan hanya untuk efek visual murahan (seperti kursor yang dikelilingi bintang).

## 3. Revolusi V8 & Node.js (2008 - 2012)

Google merilis browser **Chrome** dengan engine **V8** yang sangat cepat, mengubah pandangan dunia terhadap performa JavaScript.
- **Lahirnya Node.js (2009)**: Ryan Dahl membawa V8 ke luar browser, memungkinkan JavaScript berjalan di server. Ini adalah momen krusial yang melahirkan istilah **Full-stack JavaScript**.
- **Single Page Applications**: Kemunculan framework seperti AngularJS, React, dan Vue mengubah cara aplikasi web dibangun.

## 4. Modern JavaScript (2015 - Sekarang)

Rilis **ES6 (ECMAScript 2015)** membawa perubahan masif dengan fitur modern seperti `arrow functions`, `classes`, dan `promises`. Sejak saat itu, spesifikasi JavaScript diperbarui setiap tahun untuk memastikan bahasa ini tetap relevan dengan kebutuhan industri software modern.
