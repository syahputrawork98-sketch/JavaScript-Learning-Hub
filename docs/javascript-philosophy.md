# Filosofi & Esensi JavaScript

> **"Anything that can be written in JavaScript, will eventually be written in JavaScript." (Atwood's Law)**

## Mengapa JavaScript Ada?

JavaScript dirancang untuk menjadi **"The Universal Language of the Web"**. Filosofi utamanya adalah **Fleksibilitas** dan **Agilitas**. JavaScript tidak memaksa Anda menggunakan satu paradigma tertentu; ia adalah bahasa multi-paradigma yang mendukung gaya pemrograman Prosedural, Object-Oriented, dan Functional secara bersamaan.

## Konsep "The Event-Driven Engine" ⚡

JavaScript memiliki karakter unik yang membedakannya dari bahasa sistem:

### 1. Asynchronous & Non-blocking
Filosofi inti JavaScript (terutama di Node.js) adalah tidak pernah menunggu. Alih-alih berhenti saat melakukan operasi lambat (seperti membaca database), JavaScript menggunakan **Event Loop** untuk terus mengerjakan tugas lain dan kembali lagi saat tugas tersebut selesai. "Jangan menunggu Ban Berjalan berhenti; biarkan ia terus mengalir."

### 2. Prototypal Inheritance (Bukan Class Tradisional)
Meskipun sekarang memiliki kata kunci `class`, di bawah tendanya JavaScript tetap menggunakan **Prototype**. Objek tidak "dicetak" dari blueprint yang kaku, melainkan "mewarisi" perilaku dari objek lain secara dinamis. Ini memberikan fleksibilitas luar biasa untuk memodifikasi perilaku program saat sedang berjalan.

### 3. "Always Bet on JS"
JavaScript memiliki ekosistem terbesar di dunia (npm). Filosofinya adalah membangun solusi secara cepat, iteratif, dan dapat dideploy ke mana saja—dari browser, server, hingga perangkat IoT.

## Misi Kita

Misi dari perpustakaan ini adalah membawa Anda melampaui sekadar "pengguna framework" menjadi seorang **JavaScript Engineer** yang memahami cara kerja engine di balik layar dan mampu membangun aplikasi skala besar yang elegan dan performan.
