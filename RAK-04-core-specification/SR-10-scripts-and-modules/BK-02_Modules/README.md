# BK-02: ECMAScript Modules (ESM) (Clause 15.2)

> **"Kedaulatan Unit dan Antarmuka Terdefinisi."**

Buku ini membedah arsitektur **Modul** modern yang menyediakan isolasi tingkat tinggi, pemuatan asinkron, dan sistem ekspor/impor yang statis.

---

## 🏗️ Struktur Bab (Gold Standard)

- **[CH-01: Module Records and Linkage](./CH-01_ModuleLinkage/)**
  - Struktur `Cyclic Module Record`, mekanisme Impor/Ekspor, dan fase evaluasi.

## 🎯 Fokus Pembelajaran
1. Memahami perbedaan antara `Source Text Module Record` dan modul eksotis lainnya.
2. Menguasai mekanisme **Live Bindings** (Nilai impor yang selalu sinkron).
3. Mengenali tiga fase siklus hidup modul: *Parse*, *Instantiate*, dan *Evaluate*.
