# BK-01: Global Object and Fundamentals (Clause 19-20)

> **"Hukum Dasar dan Lingkungan Akar Hub."**

Buku ini membedah objek-objek fundamental yang menjadi basis bagi seluruh operasi di JavaScript, mulai dari objek Global hingga tipe data dasar seperti Boolean dan Symbol.

---

## 🏗️ Struktur Bab (Gold Standard)

- **[CH-01: The Global Object and Properties](./CH-01_GlobalObject/)**
  - Properti nilai (`NaN`, `undefined`) dan fungsi global (`eval`, `parseInt`).
- **[CH-02: Fundamental Foundations](./CH-02_Fundamentals/)**
  - Mesin internal di balik `Object`, `Function`, `Boolean`, dan `Symbol`.

---

## 🎯 Fokus Pembelajaran
1. Memahami peranan objek Global sebagai root dari seluruh resolusi identifier.
2. Menguasai metode statis pada `Object` untuk manajemen properti tingkat lanjut.
3. Mengenali `Symbol` sebagai cara untuk membuat kunci properti yang unik dan tak terlihat.
