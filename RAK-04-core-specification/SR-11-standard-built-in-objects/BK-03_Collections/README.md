# BK-03: Text, Arrays, and Collections (Clause 22-24)

> **"Pengolahan Narasi dan Struktur Data Hub."**

Buku ini membedah bagaimana Hub mengelola teks melalui **String** dan **RegExp**, serta bagaimana data diorganisir dalam urutan (**Array**) atau himpunan (**Map/Set**).

---

## 🏗️ Struktur Bab (Gold Standard)

- **[CH-01: Text Processing and Strings](./CH-01_TextProcessing/)**
  - Manipulasi karakter Unicode dan utilitas pencarian teks.
- **[CH-02: Regular Expression Matching](./CH-02_RegExp/)**
  - Mesin pencocokan pola (`RegExp`) dan penangkapan grup.
- **[CH-03: Indexed and Keyed Collections](./CH-03_IndexedKeyed/)**
  - Sirkuit Array, Map, Set, dan varian Weak (pemuatan hantu).

---

## 🎯 Fokus Pembelajaran
1. Memahami efisiensi pencarian teks melalui algoritma `String.prototype.includes`.
2. Menguasai cara `RegExp` mengeksekusi pencocokan menggunakan mesin *backtracking*.
3. Mengenali perbedaan performa antara `Map` vs Objek biasa untuk penyimpanan kunci dinamis.
