# BK-04: Memory Optimization

> **"Kesehatan Jangka Panjang Arsitektur Memori."**

Buku ini fokus pada deteksi kebocoran energi dan praktik terbaik untuk mengoptimalkan penggunaan memori di dalam Hub, memastikan Grid tetap responsif meskipun di bawah beban kerja berat.

---

## 🏗️ Struktur Bab

- **[CH-01: Energy Leaks](./CH-01_EnergyLeaks/)**
  - Identifikasi skenario kebocoran memori umum (Globals, Timers, Closures).
- **[CH-02: Efficiency Audits](./CH-02_EfficiencyAudits/)**
  - Strategi audit, profiling, dan optimasi (Object Pooling, Lazy Allocation).

---

## 🎯 Fokus Pembelajaran
1. Mendeteksi kebocoran memori menggunakan pola pemikiran GC Roots.
2. Menguasai alat audit memori (Heap Snapshots).
3. Menerapkan disiplin sirkuit untuk menjaga Hub tetap hemat energi.
