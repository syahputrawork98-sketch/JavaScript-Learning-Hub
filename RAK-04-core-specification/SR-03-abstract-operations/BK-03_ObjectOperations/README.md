# BK-03: Object Operations (Clause 7.3)

> **"Manipulasi Struktural Objek."**

Buku ini mendaftar Abstract Operations utama yang digunakan untuk berinteraksi dengan properti objek di level spesifikasi.

---

## 🏗️ Struktur Bab (Gold Standard)

- **[CH-01: Essential Object Abstract Ops](./CH-01_EssentialObjectOps/)**
  - Mekanisme internal Get, Set, HasProperty, dan CreateDataProperty.

---

## 🎯 Fokus Pembelajaran
1. Memahami perbedaan antara `Get` (pada objek) dan `GetV` (pada primitif).
2. Mengenali bagaimana Hub memeriksa keberadaan properti melalui `HasProperty`.
3. Menguasai logika `CreateDataProperty` untuk inisialisasi objek baru.
