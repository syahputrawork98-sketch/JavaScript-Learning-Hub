# BK-02: Testing and Comparison (Clause 7.2)

> **"Logika Komparasi dan Identitas."**

Buku ini membedah algoritma internal yang digunakan JavaScript untuk membandingkan dua nilai, menentukan kesamaan, dan melakukan pengurutan sirkuit.

---

## 🏗️ Struktur Bab (Gold Standard)

- **[CH-01: Equality and Comparison Algorithms](./CH-01_EqualityAlgorithms/)**
  - Membedah IsStrictlyEqual, IsLooselyEqual, SameValue, dan SameValueZero.

---

## 🎯 Fokus Pembelajaran
1. Memahami perbedaan antara `===` (Strict) dan `==` (Loose) di level algoritma.
2. Mengenali algoritma `SameValue` yang mampu membedakan `+0` vs `-0`.
3. Memahami algoritma `SameValueZero` yang digunakan oleh `Map` dan `Set` internal.
