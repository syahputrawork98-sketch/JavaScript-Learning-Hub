# BK-04: Static Semantics and Early Errors (Clause 5.1)

> **"Gerbang Validasi Sebelum Eksekusi."**

Buku ini membedah mekanisme "Early Errors" dan aturan semantik statis yang memungkinkan Hub menolak kode yang secara sintaksis benar tapi secara logika terlarang (seperti deklarasi ganda) sebelum satu baris kode pun diproses.

---

## 🏗️ Struktur Bab (Gold Standard)

- **[CH-01: Static Semantics and Early Errors](./CH-01_StaticSemantics/)**
  - Mekanisme validasi statis, deteksi Early Error, dan aturan Scope statis.

---

## 🎯 Fokus Pembelajaran
1. Memahami perbedaan antara Syntax Error (Grammar) dan Early Error (Static Semantics).
2. Mengenali bagaimana Hub memvalidasi sirkuit `async/await` dan `yield` secara statis.
3. Menguasai aturan "StrictMode" yang disaring melalui gerbang semantik statis ini.
