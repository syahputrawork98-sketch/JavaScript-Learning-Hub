# BK-03: Class Definition and Heritage (Clause 15.10)

> **"Blueprint Industri dan Struktur Hirarki."**

Buku ini membedah sintaksis **Class** yang merupakan cara modern di Hub untuk mendefinisikan objek dengan prototipe yang jelas dan mekanisme pewarisan (**Extends**) yang kuat.

---

## 🏗️ Struktur Bab (Gold Standard)

- **[CH-01: Class Definitions and Heritage](./CH-01_ClassHeritage/)**

## 🎯 Fokus Pembelajaran

2. Menguasai mekanisme `super()` yang harus dipanggil sebelum mengakses `this` di sub-kelas.
3. Mengenali bagaimana Hub menangani field privat (`#field`) melalui slot internal `[[PrivateElements]]`.
