# BK-05: Interface Links (Clause 15.4)

> **"Bagaimana objek menerima perilaku melalui metode dan accessor."**

Buku ini memperdalam hubungan antara function objects dengan permukaan objek melalui method definitions, getters, dan setters.

---

## Struktur Bab

- **[CH-01: Method Definitions](./CH-01_MethodDefinitions/)**  
  Sintaks metode ringkas, `[[HomeObject]]`, dan `super`.
- **[CH-02: Flow Sensors](./CH-02_FlowSensors/)**  
  Getter dan setter sebagai lapisan validasi serta formatting.

---

## Fokus Audit
1. Memisahkan interface behavior dari fungsi umum.
2. Menjadikan accessor semantics sebagai topik eksplisit.
3. Menormalkan buku paralel lama menjadi jalur pendalaman resmi.
