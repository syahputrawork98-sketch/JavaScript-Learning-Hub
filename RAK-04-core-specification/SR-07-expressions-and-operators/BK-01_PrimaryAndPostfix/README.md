# BK-01: Primary and Postfix Expressions (Clause 13.1-13.4)

> **"Atom dan Identitas di dalam Sirkuit."**

Buku ini membedah unit terkecil dari ekspresi (**Primary**) serta cara Hub mengakses dan memperbarui identitas data melalui operator **Postfix**.

---

## 🏗️ Struktur Bab (Gold Standard)

- **[CH-01: Primary Units and Initializers](./CH-01_PrimaryUnits/)**
  - Identifier, This, Literal, Array/Object Initializers.
- **[CH-02: Left-Hand-Side and Update Ops](./CH-02_LHSAndUpdate/)**
  - Member access, New, Call expressions, serta Increment/Decrement (++ / --).

---

## 🎯 Fokus Pembelajaran
1. Memahami bagaimana Hub mengevaluasi `this` di berbagai konteks.
2. Menguasai mekanisme pendaftaran properti secara dinamis di dalam Object Initializers.
3. Mengenali perbedaan antara operator pre-update dan post-update di level algoritma spec.
