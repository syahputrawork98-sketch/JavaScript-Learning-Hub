# Prasyarat dan Kamus Mini - Buku 04

## Prasyarat Global
- Sudah memahami scope, function, dan reference dasar dari buku 02.
- Sudah nyaman membaca object literal dan method dasar.
- Sudah memahami call-site dasar saat method object dipanggil.

## Kamus Mini Object Model
- Own property: property milik langsung object saat ini.
- Inherited property: property yang didapat dari prototype chain.
- `[[Prototype]]`: link internal ke object parent untuk lookup property.
- Shadowing: property lokal menutupi nama sama di prototype.
- Prototype pollution: injeksi property berbahaya ke prototype bersama.
