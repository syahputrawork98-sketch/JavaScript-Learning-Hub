# 05 — Function, Array, Object, dan Built-In API

## Fungsi Rak
Rak ini membahas fungsionalitas lanjutan dari functions (first-class, higher-order), manipulasi mendalam struktur data Array & Object, koleksi Map & Set, WeakMap & WeakSet, serta built-in global API standar JavaScript.

## Cakupan Awal
- First-Class & Higher-Order Functions
- Eksplorasi Array API (map, filter, reduce, mutator vs accessor)
- Map, Set, WeakMap, dan WeakSet
- Standard Built-In API (Math, Date, JSON, Proxy, Reflect)
