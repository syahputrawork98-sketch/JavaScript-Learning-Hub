# Pusat Perpustakaan Pembelajaran JavaScript

Selamat datang di `materi/`, pusat perpustakaan pembelajaran JavaScript yang baru, rapi, dan terstruktur.

---

## 1. Misi & Tujuan `materi/`
Folder `materi/` dirancang untuk menjadi wadah utama dari seluruh catatan belajar, dokumentasi, studi kasus, dan cheatsheet pemrograman JavaScript pribadi. 

Seluruh isi di dalam folder ini dibangun ulang dengan mengikuti pola kerapian visual dan tata kelola profesional dari referensi struktur proyek yang mapan, namun dengan klasifikasi domain ilmu yang murni disesuaikan untuk ekosistem JavaScript.
