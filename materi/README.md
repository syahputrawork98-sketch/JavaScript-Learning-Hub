# Pusat Perpustakaan Pembelajaran JavaScript

Selamat datang di `materi/`, pusat perpustakaan pembelajaran JavaScript yang baru, rapi, dan terstruktur.

---

## 1. Misi & Tujuan `materi/`
Folder `materi/` dirancang untuk menjadi wadah utama dari seluruh catatan belajar, dokumentasi, studi kasus, dan cheatsheet pemrograman JavaScript pribadi. 

Seluruh isi di dalam folder ini dibangun ulang dengan mengikuti pola kerapian visual dan tata kelola profesional dari PostgreSQL Knowledge Base, namun dengan klasifikasi domain ilmu yang murni disesuaikan untuk ekosistem JavaScript.

---

## 2. Struktur Rak Pembelajaran (00 - 17)
Materi pembelajaran dipecah menjadi 18 rak bertahap:
* `00-index-dan-jalur-belajar`
* `01-orientasi-sejarah-dan-fondasi-javascript`
* `02-javascript-core-language`
* `03-scope-closure-this-dan-execution-context`
* `04-object-prototype-class-dan-inheritance`
* `05-function-array-object-dan-built-in-api`
* `06-async-promise-event-loop-dan-concurrency`
* `07-dom-browser-api-dan-web-platform`
* `08-modules-package-json-npm-dan-tooling`
* `09-nodejs-runtime-dan-server-side-javascript`
* `10-error-handling-debugging-dan-testing`
* `11-performance-memory-dan-engine-internals`
* `12-security-dan-safe-javascript`
