# 08 — Modules, Package.json, NPM, dan Tooling

## Fungsi Rak
Rak ini membahas modularitas sistem di JavaScript (CommonJS vs ES Modules), pemahaman mendalam struktur `package.json`, manajemen paket/dependensi dengan npm/yarn/pnpm, serta arsitektur tooling ekosistem (transpiler, bundler, linter,formatter).

## Cakupan Awal
- Modularitas: CommonJS (require) vs ES Modules (import/export)
- Anatomi `package.json` & Semantic Versioning (SemVer)
- Pengelolaan Dependensi (NPM, Yarn, PNPM)
- Tooling Ekosistem (Vite, Babel, ESLint, Prettier)
