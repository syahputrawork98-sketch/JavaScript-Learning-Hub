# 04 — Object, Prototype, Class, dan Inheritance

## Fungsi Rak
Rak ini membahas paradigma Object-Oriented Programming (OOP) ala JavaScript, sistem pewarisan berbasis prototype (prototypal inheritance), sintaksis ES6 classes, property descriptors, serta teknik encapsulation.

## Cakupan Awal
- Prototypal Chain & Pewarisan Prototype
- Pembuatan Object (Factory, Constructor, Object.create)
- ES6 Classes, Static Methods, & Getters/Setters
- Encapsulation & Private Fields (`#`)
