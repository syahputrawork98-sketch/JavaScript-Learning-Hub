# 11 — Performance, Memory, dan Engine Internals

## Fungsi Rak
Rak ini membahas optimalisasi performa JavaScript tingkat tinggi, cara kerja internal JS engine (V8, JavaScriptCore, Spidermonkey), kompilasi JIT (Just-In-Time), mekanisme pengelolaan memori (Garbage Collection), serta deteksi dan pencegahan kebocoran memori (memory leaks).

## Cakupan Awal
- Arsitektur V8 Engine (Ignition Interpreter & TurboFan Compiler)
- Pengelolaan Memori (Stack vs Heap, Value vs Reference)
- Garbage Collection (Mark-and-Sweep, Generational GC)
- Pendeteksian Memory Leaks & Profiling Performa
