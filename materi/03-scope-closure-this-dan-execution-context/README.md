# 03 — Scope, Closure, This, dan Execution Context

## Fungsi Rak
Rak ini membedah mekanisme runtime penting JavaScript seperti global/local scope, lexical scope, closures, lexical environment, keyword `this` (serta call/apply/bind), serta konsep execution context dan execution stack.

## Cakupan Awal
- Scope Chain & Lexical Scope
- Konsep Closures & Retensi Memori
- Keyword `this` & Context Binding (Explicit, Implicit, Arrow Functions)
- Execution Context Lifecycle (Creation Phase & Execution Phase)
