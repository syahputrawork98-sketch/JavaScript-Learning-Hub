# 09 — Node.js Runtime dan Server-Side JavaScript

## Fungsi Rak
Rak ini membahas runtime Node.js, arsitektur event-driven non-blocking I/O yang ditenagai Libuv, eksplorasi Node.js core modules, serta dasar-dasar pengembangan aplikasi di sisi server (server-side).

## Cakupan Awal
- Arsitektur Node.js & Peran Libuv
- Eksplorasi Core Modules (fs, path, events, process, buffer)
- HTTP Server & Dasar Penanganan Request/Response
- Pengenalan REST API sederhana (Express/Fastify)
