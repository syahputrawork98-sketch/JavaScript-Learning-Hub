# 12 — Security dan Safe JavaScript

## Fungsi Rak
Rak ini membahas celah keamanan utama pada aplikasi JavaScript (XSS, CSRF, Prototype Pollution), praktik terbaik menulis kode aman, penggunaan Strict Mode, serta pengelolaan dan pemindaian dependensi dari kerentanan keamanan.

## Cakupan Awal
- Kerentanan JavaScript: XSS (Cross-Site Scripting), CSRF, dan Clickjacking
- Prototype Pollution & Regular Expression Denial of Service (ReDoS)
- Secure Coding Practices (Strict Mode, Sanitization, Eval Prevention)
- Dependency Security Audit (npm audit, snyk)
