# 06 — Async, Promise, Event Loop, dan Concurrency

## Fungsi Rak
Rak ini menyelami pemrograman asinkron di JavaScript, siklus hidup Promise, sintaksis async/await, mekanisme concurrency model, cara kerja Event Loop, perbedaan Microtask vs Macrotask queue, serta pengenalan multithreading/workers.

## Cakupan Awal
- Sejarah Asynchrony & Callback Hell
- Promise API & Methods (all, race, allSettled, any)
- Async/Await & Penanganan Error Asinkron
- Anatomi Event Loop, Call Stack, Task Queue, dan Microtask Queue
