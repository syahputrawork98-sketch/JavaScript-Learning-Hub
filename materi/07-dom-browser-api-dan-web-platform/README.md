# 07 — DOM, Browser API, dan Web Platform

## Fungsi Rak
Rak ini membahas interaksi JavaScript dengan runtime browser, manipulasi DOM (Document Object Model) tree, penanganan event (bubbling & capturing), penyimpanan browser, serta integrasi modern Web APIs.

## Cakupan Awal
- DOM Tree Manipulation & Traversal
- Event Lifecycle (Bubbling, Capturing, & Event Delegation)
- Browser Storage (Cookies, Local/Session Storage, IndexedDB)
- Web Platform APIs (Fetch, History API, Geolocation, Web Storage)
