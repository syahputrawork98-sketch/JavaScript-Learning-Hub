# SR-03: Built-ins

> **"Standardized Components: The Pre-built Modules of the Web Energy Hub."**

JavaScript menyediakan sekumpulan objek dan fungsi bawaan (*built-in objects*) yang sudah siap pakai untuk mengelola data, waktu, dan struktur informasi tanpa perlu membangunnya dari nol.

1. **[BK-01: Fundamental Objects](./BK-01_FundamentalObjects/README.md)**: Object, Function, Boolean.
2. **[BK-02: Numbers & Dates](./BK-02_NumbersDates/README.md)**: Number, BigInt, Math, Date.
3. **[BK-03: Text Processing](./BK-03_TextProcessing/README.md)**: String manipulation.
4. **[BK-04: Indexed Collections](./BK-04_IndexedCollections/README.md)**: Arrays and TypedArrays.
5. **[BK-05: Keyed Collections](./BK-05_KeyedCollections/README.md)**: Map and Set.
