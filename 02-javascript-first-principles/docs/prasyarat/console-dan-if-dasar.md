# Prasyarat: Console dan If Dasar

Dokumen ini untuk review cepat sebelum masuk materi Foundations awal.

## Tujuan
- Bisa menampilkan output dengan `console.log`.
- Bisa membuat percabangan sederhana dengan `if`.

## Contoh Ringkas
```js
const score = 75;
console.log('Score:', score);

if (score >= 70) {
  console.log('Lulus');
} else {
  console.log('Belum lulus');
}
```

## Bedah Singkat
1. `console.log` dipakai untuk cek nilai saat belajar/debug.
2. `if` menjalankan blok sesuai kondisi boolean.
3. Pola ini akan sering dipakai di semua topik.

## Checklist Cepat
- [ ] Bisa membaca output dari `console.log`.
- [ ] Bisa menulis `if/else` sederhana.
